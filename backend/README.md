# ConectaUNT - backend

Servicio de backend para el desarrollo de la aplicación web ConectaUNT.

## Configuración

1. Variables de entorno

- Crear un archivo `.env` en la raíz del proyecto con las siguientes variables de entorno:

```plaintext
PORT=8000
```

1. Ejecutar servidor

- Para el modo de desarrollo ejecute:

```bash
npm run start:dev
```
